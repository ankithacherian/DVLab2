Excercise 24:

Output for the Interpolate example (and why)

When d3.interpolate([20, 40, 4], [1, 12, 10]) is executed,an 'interpolater' is returned in variable 'intr'.
intr(0.5) evaluates the 2 given arrays and returns an intermediate value between the corresponding values in
array [20,40,4] and Array [1,12,10].
intr(0.5)=[10.5,26,7] (half of each element in 2 arrays).
Note: The return value is in domain [0,1] where value returned is equivalent to a=0 and b=1.>br>
